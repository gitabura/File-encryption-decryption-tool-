# 🔐 File Encryption & Decryption Tool

## 📌 Project Overview

The File Encryption & Decryption Tool is a web-based cybersecurity project that allows users to encrypt and decrypt text files using a password. It is designed to demonstrate basic file protection concepts through a simple, user-friendly interface.

This project is built using HTML, CSS, and JavaScript and runs entirely in the browser without requiring any server or database.

---

## ✨ Features

- 📂 Upload text files (.txt)
- 🔒 Encrypt files using a password
- 🔓 Decrypt encrypted files with the same password
- 📥 Download encrypted and decrypted files
- 📱 Responsive design for mobile and desktop
- 🎨 Clean and modern user interface

## 🚀 How to Use

1. Open the application in your browser.
2. Click **Choose File** and select a text (.txt) file.
3. Enter a password.
4. Click **Encrypt** to encrypt the file.
5. Download the encrypted file.
6. To decrypt, upload the encrypted file.
7. Enter the same password.
8. Click **Decrypt** and download the decrypted file.
